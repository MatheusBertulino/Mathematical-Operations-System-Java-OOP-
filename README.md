# 🧮 Mathematical Operations System (Java OOP)

## 📖 About

This project was developed in **Java** to demonstrate Object-Oriented Programming concepts such as abstraction, inheritance, and exception handling.

It simulates basic mathematical operations using an abstract class and includes a custom exception for division by zero.

---

## 🚀 Features

* Abstract class implementation
* Inheritance with subclasses
* Method overriding
* Custom exception handling
* Safe division with error handling

---

## 🛠️ Technologies

* Java
* Eclipse IDE
